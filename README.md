# 安装node

```shell
node -v
npm -v
```



# Hexo官网

[Hexo官网](https://hexo.io/zh-cn/)

# 安装Hexo

```shell
npm install -g hexo
```

# 初始化

```shell
hexo init
```



# 命令

```shell
#清空编译后的文件
hexo clean

#编译文件
hexo g

#启动服务
hexo s 

#部署网站
hexo d
```
